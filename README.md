# TinyApp Project

TinyApp is a full stack web application built with node and Express/EJS!

## Dependancies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie session

## Getting Started

- Install all dependancies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
